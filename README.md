This project builds a Convolutional Neural Network (CNN) using TensorFlow to classify images of cats and dogs from the Cats vs Dogs dataset. The dataset is automatically loaded using TensorFlow Datasets and split into 80% training and 20% validation data. All images are resized to 150×150 pixels and normalized before training. The model consists of three convolutional layers with max pooling, followed by fully connected layers and dropout to reduce overfitting. It is compiled using the Adam optimizer and binary crossentropy loss, and trained for 10 epochs to achieve binary classification between cats and dogs.
